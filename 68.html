Description

The omebf tool allows Bio-Formats to be used with an OME installation to import all supported formats into OME on the server side. However, it is critical that the OME-XML Bio-Formats produces is well-formed and correct, or else the import will fail. By testing omebf with all of our sample data, and fixing any bugs we find in the OME-XML generation, we will be improving both the OME server-side import mechanism itself, as well as the OME-XML conversion in general.
