Some formats take a long time to initialize

Reported by: melissa Owned by: melissa
Priority: major Milestone: bio-formats-4.2
Component: bio-formats Severity: major
Keywords: Cc: Robert.Kittel@…
Blocked By: Blocking:

Description

Files in some formats - notably Slidebook and Zeiss ZVI - take a very long time to initialize.

A ~600 MB Slidebook file (slidebook/kate/ax2-egcr.sld) takes 2.5 minutes to initialize.

A ~230 MB ZVI file (zeiss-zvi/vladan/vladan.zvi) takes 1.5 minutes to initialize.

It would be nice if we could reduce the initialization times for all files to under a minute.
