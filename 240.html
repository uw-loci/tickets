Description

Currently, the OMERO reader (and corresponding ImageJ plugin) only works with OMERO 2.2. We should update the reader to work with OMERO 2.3 (which uses a different API), ideally preserving connectivity to 2.2 servers. The solution to this problem should be flexible enough to allow for future OMERO upgrades.
