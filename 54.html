Description (last modified by curtis)

It would be really great to create screencasts (see the  Ruby on Rails screencasts for examples) for LOCI software tools demonstrating how to install and use them. We could even potentially cover how to develop software using our Java libraries.

Note that there is an  OME wiki page for video presentations, and a  newer HOWTO page with a few OME movies.

On Linux,  recordMyDesktop is the way to go (for Ubuntu, use package gtk-recordmydesktop).

On Mac OS X,  Snapz Pro X or  iShowU are good but not free.  Copernicus is an up-and-coming free program but still too buggy to really use. iMovie is a good application to use for encoding the final video.

On Windows, there is  Wink.
