Description

When checking the "Render as a volume" checkbox, nothing is done to the transparency to make the visualization more effective. Better would be to toggle the transparency type to "Curved," then utilize a heuristic to guess at a good transparency level. If the transparency type is already set to Curved with a non-100% value, this guessing step could be skipped.
