Keywords: Cc:

Description

A feature that would help with the interpretation of the data is if there was a way to filter the image by multiple parameters at once. It's often helpful to see which pixels have, for instance, lifetimes that are between 2 to 3 ns AND have ChiSq between 0.9 to 1.1 AND have F2 values between 0.6 to 0.8. The filter thresholds could be set with the histograms of each value.

See Jimmy if there are questions.
