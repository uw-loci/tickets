SLIM Plugin currently has no 'estimate Z from pre-fit' option as in TRI2. SPC Image estimates the same Z (offset) for single, double, and triple exponential.

For single and double exponential fits SPC Image does a great job of estimating fitted parameters. The default setting for subsequent iterative fitting is only 5 iterations. In SLIM Plugin and TRI2 the double exponential fit estimate is based on a formula using the single exponential fit estimate parameters.
