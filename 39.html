Dimension labels cannot be overridden

Reported by: curtis Owned by: curtis
Priority: critical Milestone: visbio3.40
Component: visbio Severity: serious
Keywords: Cc:
Blocked By: Blocking:

Description (last modified by curtis)

VisBio used to allow the user to manually specify the label for each dimensional axis. These labels are crucial for identifying how each axis should behave (e.g., time should be able to be animated). But since adding partial support for internal axes, such auto-detected axis labels can no longer be overridden. This feature must be restored for the program to be functional.

This problem will probably disappear once VisBio has been reworked to lean on Bio-Formats more heavily (ticket #35).
