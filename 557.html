Description

When opening a dataset with channel-specific LUTs as a virtual stack, all channels end up with the same LUT applied. For example, opening series 3 ("xy4z 2.52step 5CH") of leica-lif/vytas/data samples.lif as a virtual stack results in all channels appearing green.
