Description

VisBio can work with large (e.g., 4096x4096) images, but it is quite slow. Displaying them takes quite some time. Ideally, VisBio would have strategies to intelligently resample images to speed up display transformations in some cases. More generally, there should be a thorough investigation of how to speed up and improve VisBio's support for large images.
