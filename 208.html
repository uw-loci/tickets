Add information about format versions to web site

Reported by: curtis Owned by: melissa
Priority: minor Milestone:
Component: docs Severity: non-issue
Keywords: Cc:
Blocked By: Blocking:

Description

The Bio-Formats web site should provide information about which file format features are supported for each format. For example, the QuickTime information pop-up lists supported codecs. We should elaborate on the status of each format's support, since there are multiple versions of many formats.

In particular, we should elaborate on the following formats:

  • Imaris: include information and links for the HDF5 variant
  • ND2: information about different versions
  • OME-XML and OME-TIFF: 2007-06 version
  • AVI: supported codecs
  • supported codecs for any other container formats
  • any other formats with multiple significantly distinct versions
