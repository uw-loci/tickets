Description (last modified by curtis)

While the OME Metadata Editor is a full-featured graphical OME-XML editor, its template engine has fundamental limitations. In particular, although any subset of OME-XML can be included within the template, the on-screen arrangement of the information is not configurable.

The OME Notes template engine must be flexible enough to match what users would have in their lab books, with each field intelligently mapped to the underlying OME-XML as appropriate. Custom fields should appear in the XML as instances of a generic semantic type NotesField with name and value attributes.

See ticket #73 and #74 for further details.
