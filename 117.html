Reported by: melissa Owned by: melissa
Priority: major Milestone:

Keywords: Cc:
Blocked By: Blocking:

Description

This can be tested by running "inf -omexml <file>" - if there are invalid characters, and exception will be thrown, and the XML will not be printed.
