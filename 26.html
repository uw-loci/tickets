Description

The interactive color scale is a good start toward more fine-grained color control, but it is missing some key features:

  • It needs Load LUT and Save LUT buttons, and dropdown LUT presets menu, to be fully functional.

  • The color label should only be "tau" when "lifetimes" color mode is selected. For "counts" color mode the label should be "count."
