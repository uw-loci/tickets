Description

To reproduce:

  • Select "Plugins > LOCI > Bio-Formats Importer" in ImageJ.
  • Select "data/andor/james/Protocol1.tif"
  • Set the "View stack with" option to "Data Browser"
  • Select "OK" in the "Bio-Formats Import Options" window
  • Select "OK" in the "Bio-Formats File Stitching" window
  • Click the "Animate" button - note that the image and the caching bar show that the images being loaded are 0, 1, 0, 2, 0, 3, 0, 4, etc.
  • Click the "Stop" button
  • Manually scroll through the timepoints, and note that the image and caching bar are not updated (even after waiting several minutes).
