Description (last modified by curtis)

We need a graphical template editor to go along with the OME Notes template engine described in ticket #72. Users should be able to create and edit Notes templates ("design my notebook") with an easy-to-use graphical interface. Users should be able to easily design digital approximations of their physical lab books, including at minimum all textual parameters, with fields unmappable to OME-XML using a generic NotesField ST.
